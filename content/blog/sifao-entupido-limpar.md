# Sifão Entupido: Como Limpar Atendimento prioritário (Guia Completo)

**Água não escoa no lavatório? Mau cheiro vem do ralo?** O sifão entupido é uma das causas mais comuns. A boa notícia: **limpar sifão é fácil** e leva apenas **10 minutos** sem ferramentas especiais. Neste guia completo, explicamos passo a passo como desmontar, limpar e remontar o sifão.

## 🔍 O Que É um Sifão?

### Função

**Sifão** = Tubo em forma de U ou S sob o lavatório/banheira

**Funções principais**:
1. **Barreira água**: Retém água que bloqueia gases esgoto
2. **Filtro**: Retém objetos (anéis, brincos, cabelo)
3. **Fácil limpeza**: Desmonta sem ferramentas

### Tipos de Sifão

**Sifão garrafa** (mais comum):
- Forma cilíndrica
- Fácil desmontar
- Acumula mais sujidade

**Sifão tubo** (em U):
- Tubo curvo simples
- Menos comum em lavatórios
- Mais em banheiras

**Sifão plano** (moderno):
- Compacto, poupa espaço
- Móveis baixos
- Mais difícil limpar

## 🚨 Sinais de Sifão Entupido

### Sintomas

✅ **Água escoa devagar** (mais comum)
✅ **Mau cheiro** (esgoto, podre)
✅ **Gorgolejar** ao esvaziar
✅ **Água volta** (entupimento grave)
✅ **Moscas/insetos** perto do ralo

### Causas Comuns

**Lavatório casa banho**:
- Cabelo (70%)
- Sabão acumulado
- Pasta dentes
- Fio dental

**Lavatório cozinha**:
- Gordura solidificada
- Restos comida
- Café/chá
- Óleo (nunca deitar!)

**Banheira**:
- Cabelo (90%)
- Sabão/champô
- Produtos capilares

## 🛠️ Como Limpar Sifão (Passo a Passo)

### Ferramentas Necessárias

✅ Balde ou bacia
✅ Panos velhos
✅ Luvas borracha (opcional)
✅ Escova velha ou escova dentes
✅ Saco plástico (lixo)

**Não precisa**: Chave inglesa, alicate (maioria sifões)

### Passo 1: Parranjo (1 minuto)

1. **Coloque balde** sob o sifão
2. **Proteja chão** com panos velhos
3. **Vista luvas** (opcional mas recomendado)
4. **Abra janela** (cheiro desagradável)

### Passo 2: Desmontar Sifão (2 minutos)

**Sifão garrafa** (mais comum):
1. Segure corpo sifão com uma mão
2. Rode base no sentido anti-horário
3. Base desenrosca à mão (sem ferramentas)
4. Baixe base devagar (água vai sair)
5. Deixe água escorrer para balde

**Sifão tubo**:
1. Desaperte porcas nas duas extremidades
2. Geralmente à mão ou chave inglesa
3. Retire tubo completo

**Dica**: Se muito apertado, use pano para melhor aderência

### Passo 3: Esvaziar e Inspecionar (1 minuto)

1. **Esvazie água** do sifão no balde
2. **Observe conteúdo**:
 - Cabelo emaranhado
 - Gordura solidificada
 - Objetos perdidos (anéis, brincos!)
 - Restos comida

3. **Guarde objetos valiosos** encontrados

### Passo 4: Limpar Sifão (3 minutos)

**Limpeza básica**:
1. Retire sujidade grande com mãos (luvas!)
2. Despeje no saco plástico
3. Escove interior com escova velha
4. Enxague bem com água corrente

**Limpeza profunda** (se muito sujo):
1. Mergulhe em água quente + detergente
2. Deixe 5-10 minutos
3. Escove vigorosamente
4. Enxague até água sair limpa

**Gordura teimosa** (cozinha):
1. Água fervente + detergente desengordurador
2. Ou bicarbonato + vinagre
3. Deixe atuar 10 minutos
4. Escove e enxague

### Passo 5: Limpar Tubagem (2 minutos)

**Enquanto sifão desmontado**:
1. Olhe para cima (tubo parede)
2. Retire cabelo/sujidade visível
3. Use escova longa ou arame (cuidado)
4. Enxague com água

**Ralo lavatório**:
1. Retire tampa ralo (se possível)
2. Limpe cabelo acumulado
3. Escove bordas

### Passo 6: Remontar Sifão (1 minuto)

1. **Verifique vedantes** (anéis borracha)
 - Se gastos/rachados: substituir (€2-5)
 - Se OK: reutilizar

2. **Posicione base** no corpo sifão
3. **Aperte à mão** no sentido horário
4. **Aperte firme** mas não excessivo
5. **Não use ferramentas** (pode partir)

### Passo 7: Testar (30 segundos)

1. **Abra torneira** (água fria)
2. **Observe fugas** no sifão
3. **Se pinga**: Aperte mais um pouco
4. **Se continua**: Verificar vedante

**Teste completo**:
- Encha lavatório
- Retire tampa
- Observe escoamento rápido
- Sem fugas

## 🔧 Problemas Comuns

### "Não Consigo Desmontar"

**Causas**:
- Calcário colou peças
- Rosca oxidada
- Apertado demais

**Soluções**:
1. Aplique WD-40 ou vinagre (30 min)
2. Use pano para melhor aderência
3. Chave inglesa (cuidado não partir)
4. Se não consegue: chame profissional

### "Continua Entupido Após Limpar"

**Causa**: Entupimento está na tubagem (não no sifão)

**Soluções**:
1. [Desentupir com métodos profissionais](/blog/entupimento-grave-solucao)
2. Mola desentupidora
3. Hidrojato
4. Chamar canalizador

### "Fuga Após Remontar"

**Causas**:
- Vedante gasto
- Não apertou suficiente
- Rosca danificada

**Soluções**:
1. Aperte mais (à mão)
2. Substitua vedantes (€2-5)
3. Se rosca partida: substituir sifão (€15-30)

### "Mau Cheiro Continua"

**Causas**:
- Sifão sem água (evaporou)
- Ventilação insuficiente
- Problema na coluna esgoto

**Soluções**:
1. Deite água no ralo (encher sifão)
2. Use regularmente (evita evaporação)
3. Se persiste: verificar ventilação

## 🏠 Manutenção Preventiva

### Frequência Limpeza

**Lavatório casa banho**: 6-12 meses
**Lavatório cozinha**: 3-6 meses
**Banheira**: 6-12 meses
**Duche**: 3-6 meses (mais cabelo)

### Prevenção Entupimentos

**Casa banho**:
✅ Grelha ralo (retém cabelo) - €3-8
✅ Limpar cabelo do ralo semanalmente
✅ Não deitar fio dental

**Cozinha**:
✅ **NUNCA** deitar óleo/gordura
✅ Limpar pratos antes lavar
✅ Água quente após lavar louça
✅ Bicarbonato + vinagre mensal

**Geral**:
✅ Água quente semanal (dissolve gordura)
✅ Usar sifão regularmente (evita evaporação)

### Limpeza Rápida Mensal

**Sem desmontar** (5 minutos):
1. Tape ladrão (overflow)
2. Encha lavatório água quente
3. Adicione detergente
4. Deixe escoar de uma vez
5. Pressão limpa sifão

**Ou**:
1. 1/2 chávena bicarbonato no ralo
2. 1 chávena vinagre
3. Tape 15 minutos
4. Água fervente para enxaguar

## ❓ Perguntas Frequentes

### Como limpar sifão?
Desmontar base (rosca à mão), esvaziar, limpar com escova, enxaguar, remontar. **10 minutos**, sem ferramentas.

### Com que frequência limpar?
**Casa banho**: 6-12 meses. **Cozinha**: 3-6 meses. Mais frequente se água dura ou uso intenso.

### Preciso ferramentas?
**Não**. Maioria sifões desmonta à mão. Só precisa balde, panos, luvas (opcional).

### Quanto custa?
**DIY**: €0-5. **Profissional**: €40-60 (limpeza) ou €60-100 (substituição).

### Sifão continua entupido após limpar?
Entupimento está na **tubagem** (não no sifão). Precisa [desentupimento profissional](/blog/entupimento-grave-solucao) com mola ou hidrojato.

### Como evitar entupimentos?
**Grelha ralo** (€3-8), limpar cabelo semanalmente, nunca deitar óleo, água quente semanal, bicarbonato+vinagre mensal.
