# Contador de Água Disparado: Fuga Escondida ou Erro Leitura?

**Conta água €200-500 vs €30-50 normal? Contador rodou 100m³ num mês?** Contador disparado indica **fuga escondida (80% casos)**, erro contador (15%) ou fraude (5%). Neste guia completo, explicamos como verificar contador, detetar fugas escondidas com câmara térmica FLIR, reclamar erro e custos arranjo €80-600.

## 🔧 Como Verificar Contador

### Teste Fuga (5 minutos)

**Passo a passo**:
1. **Fechar todas torneiras**
   - Cozinha, casa banho, exterior
   - Máquinas lavar (não usar)
   - Esquentador/caldeira (não usar)

2. **Anotar leitura contador**
   - Números pretos (m³)
   - Ponteiros vermelhos (litros)
   - Fotografar

3. **Esperar 1-2 horas**
   - Não usar água
   - Ninguém usa

4. **Verificar contador novamente**
   - Se mudou = **FUGA**
   - Se igual = OK

**Resultado**:
- Mudou muito (10+ litros) = Fuga grande
- Mudou pouco (1-5 litros) = Fuga pequena
- Não mudou = Sem fuga (ou erro contador)

### Calcular Desperdício

**Exemplo**:
- Teste 2h: Contador subiu 10L
- Por hora: 5L/h
- Por dia: 120L/dia
- Por mês: 3.6m³/mês
- **Custo**: €15-20/mês desperdiçado

### Localizar Fuga

**Passo 1**: Fechar torneira geral
- Se contador para = fuga dentro casa
- Se contador continua = fuga exterior (ramal)

**Passo 2**: Verificar óbvio
- Autoclismo (teste corante)
- Torneiras pingam
- Máquinas lavar
- Esquentador

**Passo 3**: Câmara térmica (profissional)
- Deteta fuga escondida
- Sem partir paredes
- €80-120
- **Mais eficaz**
