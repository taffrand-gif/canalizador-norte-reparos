📋 Como Escolher

  • Multicamada: mais fácil (uniões de compressão, sem soldadura)
  • PPR: mais barato, juntas mais fortes (termofusão)
  • 16mm para pontos individuais, 20-25mm para distribuição
  • Multicamada para remodelações (flexível, passa em espaços apertados)
  • PPR para obra nova (mais económico em grandes quantidades)

🏆 A Nossa Recomendação

Para remodelações: multicamada Henco (flexível, rápido). Para obra nova: PPR (mais económico). Ambos duram 50+ anos.
