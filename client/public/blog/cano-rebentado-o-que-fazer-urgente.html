Cano Rebentado: O Que Fazer Urgentemente

Um cano rebentado é uma emergência. Cada segundo conta para minimizar danos de água.

🔍 Causas Mais Comuns

1. Congelamento

Sintoma: Cano rebenta no inverno.

Solução: Descongelar lentamente com ar quente.

2. Pressão Excessiva

Sintoma: Cano rebenta de repente.

Solução: Instalar redutor de pressão.

3. Corrosão

Sintoma: Cano velho, ferrugem.

Solução: Substituir cano.

4. Golpe de Aríete

Sintoma: Batida forte ao fechar torneira.

Solução: Instalar amortecedor de golpe de aríete.

🛠️ O Que Pode Fazer Sozinho

  1. FECHAR VÁLVULA GERAL IMEDIATAMENTE
  2. Desligar eletricidade nessa zona
  3. Colocar baldes para recolher água
  4. Abrir torneiras para drenar água residual
  5. Fotografar danos para seguro
