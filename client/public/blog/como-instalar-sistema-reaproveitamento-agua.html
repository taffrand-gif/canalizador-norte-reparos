Como Instalar Sistema Reaproveitamento Água

Reaproveitar água da chuva para rega, autoclismos e lavagens poupa até 50% na conta de água. Em Trás-os-Montes, a precipitação é suficiente para ser viável.

Passo 1: Calcular Necessidades

Rega: ~5L/m²/dia. Autoclismos: ~30L/pessoa/dia. Máquina lavar: ~50L/lavagem. Dimensione o depósito para 2-4 semanas.

Passo 2: Instalar Depósito

Enterrado (melhor) ou à superfície. Mínimo 1000L para rega, 3000-5000L se incluir autoclismos. Com filtro na entrada.

Passo 3: Ligar às Caleiras

Desvie a água das caleiras do telhado para o depósito. Instale filtro de folhas e first-flush (descarta os primeiros litros sujos).

Passo 4: Sistema de Distribuição

Bomba submersível no depósito. Circuito separado (NUNCA misture com água potável). Identifique torneiras com 'Água não potável'.

Passo 5: Overflow e Manutenção

Tubo de overflow para o esgoto pluvial. Limpe filtros 2x/ano. Verifique bomba anualmente.
