Como Instalar Banheira

Instalar uma banheira requer planeamento cuidadoso. O peso cheio (200-300kg) exige chão resistente e as ligações devem ser acessíveis para manutenção.

Passo 1: Verificar o Chão

O chão deve suportar o peso da banheira cheia + pessoa (~300kg). Em pisos de madeira, pode ser necessário reforçar.

Passo 2: Posicionar

Coloque a banheira no local. Nivele com os pés ajustáveis. A banheira deve ter ligeira inclinação para o ralo.

Passo 3: Ligar Esgoto

Instale o sistema de escoamento (ralo + overflow). Ligue ao tubo de esgoto com sifão. Todas as juntas com cola PVC.

Passo 4: Ligar Água

Traga tubos de água quente e fria até à posição da torneira. Use mangueiras flexíveis para a ligação final.

Passo 5: Impermeabilizar e Revestir

Aplique silicone em todas as juntas parede-banheira. Se usar avental, deixe acesso ao sifão para manutenção.
