    📋 Como Escolher

    • Gás MAPP é melhor que propano (mais quente, mais rápido)
    • Solda sem chumbo obrigatória para água potável
    • Flux (decapante) é essencial para boa aderência
    • Escova de aço para limpar o cobre antes de soldar
    • Proteção: luvas, óculos e protetor de chama para paredes
