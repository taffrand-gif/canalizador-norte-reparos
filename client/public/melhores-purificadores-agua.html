📋 Como Escolher o Purificador Ideal

  • Água rede pública: Carvão ativado suficiente (remove cloro)
  • Poço/furo: UV essencial (elimina bactérias)
  • Arrendamento: Jarro ou torneira (sem instalação)
  • Casa própria: Sistema fixo 3 estágios
  • Manutenção: Trocar filtros cada 6-12 meses

❓ Perguntas Frequentes

Purificador vs osmose inversa?

Purificador remove cloro, sedimentos, melhora sabor. Osmose remove 99% tudo (metais, químicos). Osmose é superior mas mais cara.

Quanto custam os filtros?

Carvão ativado: 15-25€ cada 6 meses. UV lâmpada: 40-60€ cada ano. Jarro BRITA: 5€/mês. Cerâmica: lavável, dura anos.

UV precisa de eletricidade?

Sim, lâmpada UV consome 8-15W (muito pouco). Essencial para eliminar bactérias. Ideal para água de poço.

Quanto tempo demora a instalar?

Jarro/torneira: imediato. Sistema 3 estágios: 1-2 horas. UV: 2-3 horas (ligação elétrica).

Posso instalar sozinho?

Jarro/torneira: sim. Sistema fixo: sim, com instruções. UV: recomendamos profissional (ligação elétrica).

🏆 A Nossa Recomendação

Para água rede, purificador carvão 3 estágios oferece melhor relação qualidade-preço. Para poços, UV é essencial. Jarro BRITA é prático para arrendamento.
