## Opções de Sistema de Água Quente 2026

### Esquentador a Gás (Instantâneo)
**Vantagens:**
- Água quente ilimitada
- Mais económico para uso moderado
- Instalação relativamente simples
- Bom para casas pequenas

**Desvantagens:**
- Depende de pressão da água
- Consumo de gás variável
- Necessita chaminé para evacuação
- Temperatura pode flutuar

**Preço instalação:** €300-€600 + €150-€300 instalação

### Termoacumulador Elétrico
**Vantagens:**
- Temperatura constante
- Independência da pressão da água
- Instalação simples (só precisa de tomada)
- Bom para pontos de uso isolados

**Desvantagens:**
- Consumo elétrico elevado
- Água quente limitada (depende da capacidade)
- Tempo de aquecimento (1-3 horas)
- Ocupa espaço considerável

**Preços por capacidade:**
- 30L: €200-€350 + €100-€200 instalação
- 50L: €300-€500 + €120-€250 instalação  
- 80L: €400-€700 + €150-€300 instalação
- 100L: €500-€900 + €180-€350 instalação

### Bomba de Calor para Água Quente
**Vantagens:**
- Eficiência energética muito alta (COP 3-4)
- Economia de 60-70% vs elétrico tradicional
- Funciona também como desumidificador
- Ecológico (sem emissões diretas)

**Desvantagens:**
- Investimento inicial mais elevado
- Necessita de espaço para instalação
- Produção de água quente mais lenta
- Pode fazer algum ruído

**Preço instalação:** €1.500-€3.000 incluindo instalação

### Sistema Solar Térmico
**Vantagens:**
- Custo operacional quase zero
- Ecológico (energia renovável)
- Incentivos fiscais disponíveis
- Vida útil longa (20+ anos)

**Desvantagens:**
- Investimento inicial elevado
- Depende das condições meteorológicas
- Necessita de sistema de apoio (elétrico/gás)
- Ocupa espaço no telhado

**Preço instalação:** €3.000-€6.000 para sistema familiar
